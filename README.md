# 🧩 Sudoku Game (C++)

## 📌 Overview

This project is an interactive **Sudoku Game** developed in **C++**. The application allows users to solve a predefined Sudoku puzzle by entering values in empty cells and validates the solution based on standard Sudoku rules.

The project demonstrates the use of core programming concepts such as arrays, loops, conditional statements, and validation logic, making it a strong mini project for academic purposes.

---

## 🎯 Features

* Display of a 9×9 Sudoku puzzle
* User input for filling empty cells
* Prevention of editing fixed cells
* Validation of final solution
* Clear and structured grid format
* Console-based interactive system

---

## 🛠️ Technologies Used

* **Language:** C++
* **Compiler:** MinGW (GCC)
* **IDE/Editor:** Visual Studio Code
* **Concepts Used:**

  * 2D Arrays
  * Loops & Conditions
  * Input/Output Handling
  * Validation Logic

---

## ▶️ How to Run

### Step 1: Compile the program

```bash
g++ sudoku.cpp -o sudoku
```

### Step 2: Run the program

```bash
.\sudoku.exe
```

---

## 📥 Input Format

* Enter values only for empty cells (represented as `0`)
* Follow instructions shown in the terminal

---

## 📤 Output

* Displays the Sudoku puzzle
* Accepts user input
* Shows:

  * ✅ Correct Solution
  * ❌ Incorrect Solution

---

## 🧠 Algorithm Used

The project uses **validation logic** and structured comparison to verify the correctness of the Sudoku solution.
(Advanced version can use **Backtracking Algorithm** for solving.)
